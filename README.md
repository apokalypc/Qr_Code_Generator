# QR Code Generator (Python)

## What It Does
A simple Python program that generates a QR code from text or a URL and saves it as an image.

## Features
- Converts text or URLs into QR codes
- Saves QR code as an image file
- Easy to use from the terminal

1. Make sure Python 3 is installed.
